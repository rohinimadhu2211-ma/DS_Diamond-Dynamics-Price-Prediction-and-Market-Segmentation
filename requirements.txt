Dataset Description

This dataset contains information about diamonds and their physical attributes used to predict their market price.

Columns Description
Column	Description
carat	Weight of the diamond
cut	Quality of the cut (Fair, Good, Very Good, Premium, Ideal)
color	Diamond color grade (D to J)
clarity	Measure of inclusions or internal flaws
depth	Total depth percentage
table	Width of the top of the diamond
price	Price of the diamond in USD
Dataset Use

The dataset is used for:

Predicting diamond prices using machine learning models

Identifying patterns in diamond pricing

Performing clustering for market segmentation

Model Description

This project uses regression and clustering models to analyze diamond prices and market segments.

Price Prediction Models

The following machine learning models are used:

Linear Regression

Used as a baseline model to understand the relationship between diamond features and price.

Decision Tree Regressor

Captures non-linear relationships between variables.

Random Forest Regressor

An ensemble learning method that improves prediction accuracy by combining multiple decision trees.

Model Evaluation

Models are evaluated using:

Mean Absolute Error (MAE)

Mean Squared Error (MSE)

R² Score

Market Segmentation

K-Means clustering is applied to group diamonds into segments based on similar attributes and price ranges.
